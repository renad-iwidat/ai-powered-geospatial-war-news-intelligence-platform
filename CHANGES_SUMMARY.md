## 📁 الملفات الجديدة

### 1. `scripts/cleanup_locations.py`
**الوظيفة:** تنظيف البيانات الخاطئة

**الميزات:**
- حذف 15+ موقع خاطئ
- تصحيح 25+ موقع بإحداثيات خاطئة
- حذف المواقع بإحداثيات غير معقولة
- حذف الأحداث والمقاييس المرتبطة

**الاستخدام:**
```bash
python scripts/cleanup_locations.py
```

### 2. `scripts/reprocess_news.py`
**الوظيفة:** إعادة معالجة الأخبار

**الميزات:**
- حذف الأحداث والمقاييس القديمة
- إعادة معالجة جميع الأخبار
- استخراج الأماكن بشكل صحيح
- استخراج المقاييس من جديد

**الاستخدام:**
```bash
python scripts/reprocess_news.py
```

## 📊 التأثير على الأداء

### قبل التحديث
- كل موقع يحتاج طلب Nominatim
- معدل: 1 طلب/ثانية
- قد يحدث خلط بين الأسماء المشابهة

### بعد التحديث
- المواقع الأساسية من القائمة البيضاء (فوري)
- المواقع الأخرى من Nominatim (عند الحاجة)
- لا خلط بين الأسماء المشابهة
- أسرع بـ 10x للمواقع الأساسية
