# ملخص سريع: إصلاح مشكلة المواقع الجغرافية ⚡

## 🎯 المشكلة
- "لبنان" (الدولة) يتم الخلط بينها و "بنان" (مدينة في تونس)
- إحداثيات خاطئة (خط عرض وطول غلط)

## ✅ الحل (3 خطوات)

### 1️⃣ تحديث الكود
✓ تم إضافة قائمة بيضاء (Whitelist) بـ 50+ مدينة ودولة أساسية
✓ تم تحديث دالة `geocode_best_effort()` لاستخدام القائمة أولاً
✓ الملف: `app/services/geo/location_processor.py`

### 2️⃣ تنظيف قاعدة البيانات
```bash
python scripts/cleanup_locations.py
```
- حذف 15+ موقع خاطئ
- تصحيح 25+ موقع بإحداثيات خاطئة
- حذف الأحداث والمقاييس المرتبطة

### 3️⃣ إعادة معالجة الأخبار
```bash
python scripts/reprocess_news.py
```
- حذف الأحداث والمقاييس القديمة
- إعادة معالجة جميع الأخبار
- استخراج الأماكن بشكل صحيح
